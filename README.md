# Logica-de-Programacao-
Logica de Programacao em C++
Este repositório reúne exercícios e projetos desenvolvidos ao longo do curso de Análise e Desenvolvimento de Sistemas da FATEC Mogi das Cruzes, com foco no aprimoramento da lógica de programação e na construção de fundamentos sólidos em desenvolvimento de software.

Os exercícios foram elaborados para fortalecer habilidades analíticas, resolução de problemas e raciocínio computacional, utilizando a linguagem C++ como base principal.

## Competências desenvolvidas

* Estruturas condicionais e tomadas de decisão
* Estruturas de repetição e controle de fluxo
* Desenvolvimento de funções e modularização
* Manipulação de vetores e matrizes
* Conversão e representação numérica
* Construção de algoritmos e lógica computacional
* Boas práticas de programação

## Tecnologias utilizadas

* C++
* Git e GitHub

O objetivo deste repositório é demonstrar minha evolução prática em programação, organização de código e resolução de problemas durante a graduação.
